# Typescript Starter 

## Setup Typescript 

* Windows Installation 

    ```
   npm install -g typescript 
    ```

* Mac/Linux Installation 


    ```
    sudo npm i -g typescript  
    ```

## Dependencies

* Install Express and Nodemon

    ```
    npm i -D typescript ts-node nodemon @types/node @types/express  
    ```


## Run Typescript 
    npm run dev
